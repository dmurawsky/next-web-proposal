# Next Web Proposal
A proposal for a high level, public simulation of the Next Web

Let's break this down:

- **Proposal:** Because it's not an official project yet
- **High Level:** Because it will use Backend as a Service and serverless technologies
- **Public:** Because it's for use by the general public
- **Simulation:** Because the real 'Next Web' will require a whole new type of browser
- **Next Web:** Because it will be a [semantic web of linked data](https://www.ted.com/talks/tim_berners_lee_on_the_next_web)
