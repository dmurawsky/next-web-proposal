# Next Web Proposal
A proposal for a high level, public simulation of the Next Web

Let's break this down:

- _Proposal:_ Because it's not an official project yet
- _High Level:_ Because it will use Backend as a Service and serverless technologies
- _Public:_ Because it's for use by the general public
- _Simulation:_ Because the real 'Next Web' will require a whole new type of browser
- _Next Web:_ Because it will be a [semantic web of linked data](https://www.ted.com/talks/tim_berners_lee_on_the_next_web)
